# Satellite Communication Resilience Simulation: Performance Analysis Report

## 1. Executive Summary
This report analyzes the performance of a hybrid vehicular communication system (Satellite and Ground Station) under simulated dynamic environmental conditions. The system demonstrates robust self-healing and adaptive switching capabilities.

## 2. Key Performance Indicators (KPIs)
* **Packet Delivery Ratio (PDR)**: 100% (All 20 test messages were successfully delivered).
* **Automatic Switching Mechanism**: The system executed 14 automatic satellite handovers, ensuring the connection to the lowest-latency link in real-time.
* **Ground Network Contribution**: Maintained 100% stability during simulated urban transit, effectively offloading satellite network congestion.

## 3. Technical Summary
The system exhibits high resilience through "Context Awareness" and "Fault Recovery":
- **Adaptive Retries**: Automatically initiates smart retries and link re-selection when signal interference is detected.
- **Seamless Handoff**: Transitions between Satellite and Ground networks are smooth, meeting the strict requirements for vehicular communication systems.

This analysis is derived from the `simulation_results.csv` dataset, confirming the system's reliability and robustness under stress testing.
