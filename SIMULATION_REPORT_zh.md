# 衛星通訊系統韌性模擬：效能分析報告

## 1. 系統執行摘要
本報告分析了車載混合式通訊系統（衛星與地面網路）在模擬場景下的通訊效能。系統成功展示了在極端環境下的動態切換能力。

## 2. 關鍵效能指標 (KPIs)
* **訊息成功率 (PDR)**：100% (所有訊息最終皆成功送達)。
* **自動切換機制**：系統在衛星環境下共執行 14 次自動切換，確保隨時連結至最低延遲之衛星鏈路。
* **地面模式貢獻**：在模擬城市區域時提供 100% 的穩定通訊，有效降低了衛星網路的負載。

## 3. 技術總結
系統具備優秀的「環境感知」與「錯誤恢復」能力：
- 當遭受訊號干擾時，系統會啟動智慧重試與鏈路重選。
- 在地面與衛星網路間的過渡平滑，符合車載系統設計需求。

## 4. 數據來源
本分析基於程式產生的 `simulation_results.csv` 數據集，證明了其架構在壓力測試下的可靠性。
